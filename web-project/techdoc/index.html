Introduction

Web application development involves the creation of dynamic and interactive programs that run on web servers and are accessed by users through web browsers. This field of development combines various technologies, frameworks, and programming languages. This process involves creating user-friendly applications accessible via web browsers, utilizing a combination of client-side and server-side technologies.

Client-Side Development

Client-side development involves activities that occur within the user's web browser, focusing on the presentation layer and user interaction

Technologies:

  • Hypertext Markup Language(HTML)
  • Cascading Style Sheets(CSS)
  • JavaScript

Responsibilities:

  • Creating and structuring web pages(HTML)
  • Styling and designing webpages(CSS)
  • Implementing interactive features and behavior(JavaScript)
  • Managing user interface components and user experience

Server-Side Development

Server-side development involves activities that occur on the web server, handling data processing, business logic, and interactions with databases.

Technologies:

  • PHP
  • Python
  • JavaScript
  • Java
  • MySQL

Responsibilities:

  • Managing data storage and retrieval from databases.
  • Executing business logic and application functionality.
  • Implementing user authentication and authorization.
  • Generating dynamic content to be sent to the client.

Hypertext Markup Language

HTML (HyperText Markup Language) is the standard markup language used to create the structure and content of web pages. It defines the elements and components that make up a web page, allowing developers to organize and present content on the internet.

Key Concepts

Document Structure

HTML documents follow a hierarchical structure defined by nested elements. The basic structure of an HTML document includes the <html>, <head>, and <body> elements.

Example: 

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
  </head>
  <body>
    <p>This is a paragraph</p>
  </body>
</html>

Elements

HTML documents are composed of elements, which are defined by tags enclosed in angle brackets ( < > ).

Tags

Tags are used to define different types of content within an HTML document.

Attributes

Attributes provide additional information about HTML elements and are defined within the opening tag.

Basic Tags

  • <html>: Represents the root element of an HTML document.
  • <head>: Contains meta-information about the HTML document, such as the title, character set, and links to external resources.
  • <title>: Sets the title of the HTML document displayed in the browser tab.
  • <body>: Contains the content of the HTML document that is displayed in the browser window.
  • <h1 to <h6>: Defines headings of different levels, where <h1> is the highest level and <h6> is the lowest level.
  • <p>: Represents a paragraph of text.
  • <a>: Creates hyperlinks to other web pages or resources.
  • <img>: Inserts images into the HTML document.

Semantic HTML

Semantic HTML refers to using HTML elements that convey meaning about the content they contain. This helps improve readability of the web page.

Example:

  • <header>
  • <nav>
  • <main>
  • <section>
  • <article>
  • <aside>
  • <footer>
Cascading Style Sheets

CSS (Cascading Style Sheets) is a fundamental technology used in web development to define the presentation and styling of HTML documents. It plays a crucial role in enhancing the visual appeal, layout, and overall aesthetics of web pages. CSS enables developers to control the appearance of elements on a webpage, ensuring consistency and creativity in design.

Key Concepts

Selectors

Selectors target specific HTML elements to apply styles. Common selectors include element selectors (p, h1, div), class selectors (.className), and ID selectors (#idName).

Properties and Values

CSS properties define the appearance of HTML elements, while values determine how these properties are applied. For example, color: blue; sets the text color to blue.

Inline Style Sheets

Styles applied directly to HTML elements using the style attribute.

<p style="color: red;">This is a red paragraph. </p>

Internal Style Sheets

Styles defined within <style> tags in the <head> section of an HTML document.

<style>
  h1 {
    color: green;
  }
</style>

External Style Sheets

Styles stored in separate .css files and linked to HTML documents using the <link> tag.

<link rel="stylesheet" href="styles.css">
Flexbox

Flexbox is a layout model designed to simplify the arrangement of elements within a container, providing a more efficient and responsive way to design complex web layouts.

Key Concepts

Main Axis and Cross Axis

Main Axis: The primary axis defined by the flex-direction property (row, row-reverse, column, column-reverse).
Cross Axis: The perpendicular axis to the main axis.

Flex Container

An element set to display: flex; becomes a flex container, which enables the use of flexbox properties to control the layout of its child elements.

Flex Items

Children of a flex container are referred to as flex items.

Flex Properties

Flexbox introduces a set of properties to control the behavior and layout of flex items

  • display: flex; : Establishes a flex container.
  • flex-direction : Defines the main axis direction.
  • justify-content : Aligns flex items along the main axis.
  • align-items : Aligns flex items along the cross axis.
  • flex-grow, flex-shrink, flex-basis : Controls how flex items grow, shrink, and initialize.

To create a basic flexbox layout, follow these steps:

1. Set Up the Flex Container

.container {
  display: flex;
  flex-direction: row; 
  justify-content: center; 
  align-items: center; 
}

2. Define FLex Items

.item {
  flex: 1; 
}

3. HTML Structure

<div class="container">
  <div class="item">Item 1 </div>
  <div class="item">Item 2 </div>
  <div class="item">Item 3 </div>
</div>
Window Subsystem for Linux

Windows Subsystem for Linux (WSL) is a compatibility layer developed by Microsoft that enables you to run a Linux operation system on Windows 10 or later versions.

List of Commands

Commands

  • wsl --install: installs Linux
  • wsl: opens Linux
  • logout or exit returns to command prompt in windows:
  • clear: clears the terminal screen

Apache Commands

  • sudo apt upgrade: updates Linux before installation of Apache
  • sudo apt update: upgrades Linux
  • sudo apt-get install apache2: installs Apache server
  • sudo service apache2 start: starts Apache Server
  • sudo service apache2 stop: stops Apache Server

Apache Setup

  • sudo nano: open text editor
  • sudo chown -R username path/to/working/directory: make folder writable
  • sudo nano /etc/apache2/sites-available/000-default.conf: enable site look for document root and change it your desired folder to run in localhost
  • sudo systemtl status apache2: check if apache is running
  • sudo sustemtl restart apache2: restart Apache

Folder Locations

  • Apache Folder/etc/apache2/
  • Document Root/var/www/
